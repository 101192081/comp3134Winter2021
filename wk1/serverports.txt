Q: What is the definition of server port?
A: A server port is a communication endpoint.

Q: What is server port range?
A: Server port range is the port number that a port can have, registered ports have a number from 1024 to 49151 and dynamic ports a a number from 49152 to 65535.

Q: How many bits does number above represent?
A: 10 to 14 bits

Q: State 10 common server ports

	2. 21 - file trasfer protocol command control
	3. 22 - secure shell
	4. 23 - telnet - remote login service, unencrypted text messages
	5. 25 - simple mail transfer protocol
	6. 53 - domain name system
